# Colophon

## Technologies

Built with [Next.js](https://nextjs.org/), [MDX](https://mdxjs.com/), and
[Tailwind CSS](https://tailwindcss.com/).

Deployed and hosted using [Netlify](https://www.netlify.com/).

Anonymous, GDPR-compliant, self-hosted analytics using
[Umami](https://umami.is/).

Source code available on
[GitHub](https://github.com/CharmedSatyr/personal-website).

## Art

I drew the tree logos <Tree /> in [Krita](https://krita.org/en/) using a
[Wacom Intuos Pro tablet](https://www.wacom.com/en-us/products/pen-tablets/wacom-intuos-pro).
You may notice that their colors match secondary and accent colors in both light
and dark modes.

## Icons

Brand icons are from [Simple Icons](https://simpleicons.org) and
[Tech Stack Icons](https://www.tech-stack-icons.com/). The rest are
[Heroicons](https://heroicons.com/).
